# Tutorias Java - Corte 5 | Grupo 2 DevSenior

Repositorio de ejercicios y proyectos desarrollados durante las tutorias de Java del programa DevSenior.

## Descripcion

Este repositorio contiene implementaciones practicas de conceptos fundamentales e intermedios de programacion en Java, organizados de forma progresiva desde operadores basicos hasta programacion orientada a objetos con encapsulamiento.

## Estructura del Proyecto

```
REPOSITORIO/
├── MODULO-1/                          # Fundamentos de Java
│   ├── clase-2/
│   │   └── CalculadoraCompra/         # Calculadora de compras
│   ├── clase-3/
│   │   └── techStore/                 # Sistema de gestion de ventas
│   └── clase-4/
│       └── gestionDeTienda/           # Sistema de inventario
│
└── MODULO-2/                          # POO y Encapsulamiento
    └── clase 1/
        └── encapsulamiento/           # Ejemplo de encapsulamiento
```

## Contenido por Modulos

### Modulo 1: Fundamentos y Operadores

#### Clase 2 - Calculadora de Compra
- Operadores aritmeticos y de asignacion compuesta
- Operadores logicos y ternarios
- Calculo de IVA y descuentos para clientes VIP
- Sistema de puntos y pagos en cuotas

#### Clase 3 - TechStore (Sistema de Ventas)
- Gestion de perfiles de vendedores
- Catalogo de productos tecnologicos
- Sistema de tarjetas de fidelizacion con descuentos por niveles
- Calculo de comisiones y bonificaciones
- Procesamiento de devoluciones
- Estadisticas y metricas de rendimiento

#### Clase 4 - Gestion de Tienda
- Registro de productos en inventario
- Calculo de valor total y promedio del inventario
- Aplicacion de descuentos
- Interfaz de menu interactivo

### Modulo 2: Encapsulamiento y POO

#### Clase 1 - Encapsulamiento
- Clase `Estudiante` con atributos privados
- Constructores multiples (por defecto, con parametros, completo)
- Getters y Setters con validaciones
- Metodos de logica de negocio:
  - Obtencion de calificacion segun promedio
  - Evaluacion de estado academico
  - Verificacion de elegibilidad para graduacion

## Tecnologias Utilizadas

- **Lenguaje:** Java
- **Entrada de datos:** `java.util.Scanner`
- **Paradigma:** Programacion Orientada a Objetos (POO)

## Conceptos Aplicados

- Tipos de datos primitivos y referencia
- Estructuras de control (if-else, switch, loops)
- Metodos estaticos y de instancia
- Encapsulamiento (modificadores de acceso)
- Constructores y sobrecarga
- Validacion de datos
- Aplicaciones con menu interactivo
